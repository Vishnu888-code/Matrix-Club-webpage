# 🚀 Matrix Club Website

A modern, interactive, and visually engaging **club profile website** built using HTML, CSS, and JavaScript. This project showcases a real-world tech community platform with advanced UI/UX features and smooth animations.

---

##  Features

*  Responsive and modern UI design
*  Gradient-based colorful sections
*  Smooth scroll animations
*  Animated statistics counters
*  Glassmorphism cards
*  Scroll progress indicator
*  Custom glowing cursor effect
*  3D tilt hover interactions
*  Hackathon & Events image grid
*  Animated particle background

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & animations
* **JavaScript (Vanilla JS)** – Interactivity & DOM manipulation
